🖼️ 介面組成

桌面版

  • 主地圖區域：全螢幕 MapLibre 地圖 + 圖書館標記點
  • 右上角列表面板：320px 寬，包含排序選單和圖書館列表
  • 底部資訊框：顯示圖例或選中的圖書館簡訊
  • 詳細資訊視窗：覆蓋全螢幕的模態視窗

行動版

  • 頂部搜尋欄：固定在頂部，包含搜尋框和篩選按鈕
  • 主地圖區域：響應式高度
  • 底部滑動面板：可向上展開至 70vh
  • 全螢幕詳細頁：從右側滑入

✨ 設計原則

📱 行動優先

優先考慮行動裝置使用體驗，單手即可完成所有操作。

🎯 資訊優先

座位資訊清晰可見，快速找到有空位的圖書館。

🚀 流暢體驗

所有動畫和轉場都經過優化，確保 60fps 流暢度。

♿ 無障礙

符合 WCAG AA 標準，支援鍵盤操作和螢幕閱讀器。

⚙️ 技術規格

地圖

  • 地圖引擎：MapLibre GL JS
  • 底圖：MapTiler / OpenStreetMap
  • 標記：自訂 HTML 標記 (32x32px 圓形)

前端框架

  • 建議使用：React / Vue / Svelte
  • 狀態管理：視專案需求選擇
  • 樣式方案：CSS Variables (已定義在 design-tokens.css)

響應式斷點

  • 行動版：≤ 767px
  • 平板：768px - 1023px
  • 桌面：≥ 1024px

  border-radius: var(--radius-md);
  transition: var(--transition-fast);
